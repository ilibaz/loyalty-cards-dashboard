## Overview
A simple web application that allows a user to **view and use loyalty benefits** provided by a venue *(restaurant, bar, cafÃ©, etc.)*.

## Tech Used

- **Frontend & Backend**: [Next.js](https://nextjs.org/) (React + API routes)  
- **UI Library**: React (with TypeScript)  
- **Database**: [PostgreSQL](https://www.postgresql.org/) 
- **Containerization**: [Docker](https://www.docker.com/) + Docker Compose   

> **Key Features**
> - Show list of benefits a user is eligible for  
> - Track which benefits have been used  
> - Prevent multiple uses of the same benefit  

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
